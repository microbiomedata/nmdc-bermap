    Enum: PhenotypeAssayType

    Types of phenotype assays performed by research programs. Used to catalog what phenotyping capabilities and data types each SFA/BRC generates.

    URI: nmdc_sfas_brcs:PhenotypeAssayType

    COCULTURE_INTERACTIONS None Pairwise or multi-species co-culture interaction assays
    BACTERIAL_FUNGAL_INTERACTION_ASSAY None Bacterial-fungal co-culture or proximity interaction assays
    PLANT_COLONIZATION None Plant root or leaf colonization assays
    PHAGE_HOST_RANGE None Phage-host infection range and specificity testing

    COMPETITION_ASSAYS None Competitive fitness assays between strains

    ECOFAB_PHENOTYPING None EcoFAB fabricated ecosystem phenotyping
